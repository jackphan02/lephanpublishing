MATLAB Visual Basic .Net Book illustrates the works between Visual Basic .Net programming and MATLAB. If you are a Visual Basic .Net programmer and want to use the quality mathematical built-in functions in your Visual Basic .Net applications to solve mathematical problems in VB .Net programming code, then this book is what you're looking for.

The table contents show mathematical problems solving in Visual Basic .Net code. These problems include Stand Alone Application in VB .Net, using MATLAB built-in functions in VB .Net in Console Application, and using MATLAB built-in functions in VB .Net in Window Applications .

This MATLAB Visual Basic .Net Book is in the pdf format on CD-Rom which also includes the complete Visual Basic .Net code examples of all chapters .
