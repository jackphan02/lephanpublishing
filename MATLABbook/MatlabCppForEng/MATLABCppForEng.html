MATLAB - Microsoft Visual C++ for Engineers book illustrates the works between MSVC++ programming and MATLAB. The  book describes very detail how to use MATLAB built-in functions in VB applications. The features of this book are designed to handle following topics:

  3. MSVC++ functions plot figures from MATLAB Graphics.

This book is in the pdf format and MSVC++ projects which includes the complete MSVC++ code examples of all chapters .
